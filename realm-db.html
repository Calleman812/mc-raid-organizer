How it works

This database is stored in your browser (localStorage) and persists permanently on this device.

When the Raid Organizer builds the MRT export, it looks up each player name here. If found, it appends -RealmName to their name automatically.

Players whose names already contain a - (from Raid-Helper) are expanded using the built-in abbreviation table and are not looked up here.

Moving to a new device? Use the Export button to download a .txt file, then Import it on the new browser.
